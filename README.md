Elasticsearch Two Nodes Setup with Vagrant
==============================

## Run

```sh
    berks install
```

```sh
    vagrant up
```

## Setup

### Vagrant

Install it.

### [Berkshelf](http://berkshelf.com/)

Install it.

```sh
    gem install berkshelf
```

```sh
    vagrant plugin list
```

```sh
    vagrant plugin install vagrant-berkshelf
```
